Trade & Investment Management Portal (TIMP)

Multi-lingual Admin + Public Region-segregated Demo data Static HTML prototype (GitHub Pages friendly)

What this demo shows

Event management + registration, Deal Room matchmaking, Deal Book generation, investment pipeline dashboard, virtual exhibition, searchable archive, and RBAC concepts.

What it does NOT do

No real authentication, no database, no email sending, no ATG/CRM integration. Those become the developer build items.

How to run

Unzip → run a local server (recommended) → open index.html via http://localhost:8000.

Quick demo storyline

  1. Admin creates/publishes an event and invites investors.
  2. Investor registers and gets access to materials + recordings.
  3. Deal Room recommends meetings based on sector + ticket size.
  4. Pitching module + “Request for Prospectus” interest signal.
  5. One-click Deal Book generation by country/sector.
  6. Pipeline dashboard tracks deal stage + lead “warmth”.
  7. Virtual exhibition booths for exporters / TPOs.
  8. Archive + search for audit-ready institutional memory.

Demo toggles

Language and Region are stored in your browser and apply across pages.

Tip: Set Region to Non-Regional to show “global view”. Set to a region to show segregation logic.
